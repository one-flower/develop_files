# 常用流程

1. git pull 拉取
2. git add 添加
3. git commit 提交
4. git push 同步

# 常用命令

| 命令                                 | 含义           |
| :----------------------------------- | :------------- |
| git config --global user.name "name" | 设置用户名     |
| git config --global user.email email | 设置密码       |
| git init                             | 初始化         |
| git clone (url)                      | 克隆           |
| git checkout (name)                  | 切换           |
| git pull                             | 拉取           |
| git add                              | 添加           |
| git commit                           | 提交           |
| git push                             | 上传           |
| git branch -a                        | 查看所有分支   |
| git biranc -r                        | 只查看远程分支 |
| git branch -d (name)                 | 删除分支       |
